Url per acceidr al Web Service :
url : 	http://169.254.10.113/FactorySystems/public/api/ (Local)
url : 	http://10.2.7.89/FactorySystems/public/api/ (VPN)

Funcions que es pot cridar : 

url/getErrors GET
url/getMaquinas GET
url/getTreballadors GET


url/login : 
	Params  : email, contrasenya.
	Headers : ('Accept' :  'application/json')

url/reportarError : 
	Params :$rules = [
            		"maquina"=> "required|numeric",
           		"treballador" => "required|numeric",
            		"estat_error" => "required",
            		"descripcio" => "nullable",
            		"tipus" => "required",
        	];
	Headers : ('Accept' :  'application/json')
	
	Devuelve un response status : 200 OK, 400 Bad request si algun parametro no cumple las restrucciones.
	Tambien devuelvo un json con los campos : message : String  
						  errors  : array con parametros que han causado el error.
